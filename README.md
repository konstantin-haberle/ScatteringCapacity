# Separation Capacity of Scattering Networks

This repository provides an implementation for computing the separation capacity of scattering networks.

The repository is organized as follows:

- `sc.py`: Class containing methods to compute the separation capacity of a map, as well as the number of homogeneously linearly separable dichotomies of a point set.
- `scattering_networks_finite_cyclic_groups.py`: Class implementing general scattering networks defined on finite cyclic groups.
- `examples.ipynb`: Demonstrates how to compute the separation capacity of scattering networks. It includes examples using scattering networks built from Weyl–Heisenberg and wavelet frames.
